# org-tools

Инструменты для управления структурой команды и визуализации показателей.

## Состав

### apps/
- **team-cards.html** — визуализация структуры команды: перетаскивание сотрудников между командами, импорт/экспорт Excel
- **dashboard.html** — дашборд показателей управления (в разработке)

### design/
- **design-kit.pen** — дизайн-системы в Pencil:
  - *Light & Airy* — основная дизайн-система (кнопки, карточки, метрики, прогресс-бары)
  - *Команды* — компоненты для инструмента управления командами (чипы, заголовки ролей, карточки)

## Стек
- Vanilla HTML/CSS/JS (без фреймворков)
- [SheetJS](https://sheetjs.com/) для работы с Excel
- [Lucide](https://lucide.dev/) для иконок
- [Pencil](https://pencil.ws/) для дизайна
